SnBatteryPlus - CustomBatteries pack bundle

Hey, I put this pack together to make battery progression feel better without adding a bunch of extra clutter.

Included packs:
- CopperBatteries
- GoldBatteries
- NickelBatteries
- SilverBatteries
- UraniumBatteries



Install:
1) Make sure CustomBatteries is already installed.
2) Copy the included pack folders into:
   BepInEx\plugins\CustomBatteries\Packs
3) If asked, allow overwrite.

Notes:
- This is just a curated bundle of packs for convenience.
- If anything conflicts with your existing setup, remove the pack folder causing issues.
